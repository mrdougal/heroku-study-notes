# Flows

3 types of flows in Salesforce

## Screen flow

UI that guides a user through a business process

- Quick action
- Lightning page
- "Experience cloud" site

## Auto-launched flow

Have no UI run the background

- another flow
- [Apex](./apex.md) code
- REST APIs

## Triggered flow

Auto-launched by a trigger you specify, run in the background

- time
- data changed event
- platform event
